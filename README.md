# Student Registration Form

A simple HTML form built to collect student information, including personal details, contact information, and course enrollment.

## 🚀 Features

- Full Name, Roll Number, and Date of Birth inputs
- Gender selection using radio buttons
- Email and Phone validation
- Address input using textarea
- Course selection dropdown
- Hobbies/Interests using checkboxes
- Required field validation
- Submit and Reset functionality

## 🛠️ Technologies Used

- HTML5
